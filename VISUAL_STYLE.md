# Visual Style Guide

## Pixel Art Style
- Resolution: 32x32 for agents, 16x16 for plants
- Color palette: Soft pastels (like Stardew Valley)
- Animation: 4-frame walk cycle

## Garden Layout
- Grid-based: 64x64 pixel tiles
- Sections clearly defined with fences/paths
- Background: Grass texture with subtle variation

## UI Elements
- Speech bubbles: White with black border
- Task panel: Transparent dark overlay
- Health dashboard: Top-right corner, minimal
